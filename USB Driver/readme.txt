How to use linux driver for ZTE device1.
Copy the package to your system.
Be sure of logining with root account in terminal.
2.run the command cd ../the package directory
3 chmod 777 install uninstall
4 for install, run: ./install
5 for uninstall, run: ./uninstall
note:the driver can support these distributions:Ubuntu: 11.04, 11.10, 12.04, 12.10, 13.04
Fedora: 15,16,17,18,19
uninstall file has a copy under the dir /opt.You can find it and run to uninstall our software.
